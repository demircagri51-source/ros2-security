# 🛡️ A9 - ROS2 Humble Critical/Defense Risk Analysis: SROS2, DDS-Security

**Phase 1: Starter Kit & Simulation Infrastructure**

This repository contains the simulation infrastructure and automation scripts for analyzing the cybersecurity and defense mechanisms (SROS2, DDS-Security) of a ROS2 Humble based system using a UR5e robot.

## 📌 1. Environment Details
* **OS:** Ubuntu 22.04 LTS
* **ROS Distribution:** ROS2 Humble
* **Simulator:** Gazebo Classic 11
* **Robot:** Universal Robots UR5e

## 🚀 2. Infrastructure & Automation
The simulation, control interfaces (`ros2_control`), and MoveIt2 motion planning are automated via a single bash script.

```bash
chmod +x starter_kit_baslat.sh
